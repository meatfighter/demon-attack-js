About

About This is a browser port of Demon Attack, a space shooter released in 1982 for the Atari 2600. It challenges the player to fend off waves of hovering creatures by controlling a laser cannon. The enemy patterns and weapons grow more complex with each round, culminating in foes that split into dive-bombing monsters. [ Play ] Controls The game accepts three types of input: Keyboard: Use the left and right arrow keys or the A and D keys to move. Use any other key to fire. Gamepad: Use the D-pad or analog stick to move. Press any other button to fire. Touchpad: On the configuration page, enable autofire. During play, hold down the left or right side of the screen to move in that direction. Configuration The configuration page provides three checkboxes: Autofire: The cannon continuously shoots. This is the default on touchpad-only devices. Tracer: The player steers the cannon shots. Fast: Doubles the player’s cannon speed. Differences The original game was designed and programmed by Rob Fulop, with graphics created by Michael Becker. This TypeScript port was developed through detailed observations of gameplay and analysis of the original 6507 assembly language source code. It contains all the original graphics, sans the Imagic logo. The configuration page provides checkboxes to reproduce the game variations from the original. However, the port does not support 2-player modes nor “advanced” mode (starting on wave thirteen). The port introduced autofire to simplify input on touchpad-only devices. In the original, the player reached a kill screen after 84 waves. A subsequent release patched this, allowing gameplay to continue at a plateaued difficulty. The port reproduces the latter.
